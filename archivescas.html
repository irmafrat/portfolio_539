Jan. 2020-May 2020

Collecting Tweets: Recent protest movement in Puerto Rico

University of Michigan School of Information

GitHub Repo

With this program, the user will be able to search a database of tweets related to historical events in Puerto Rico. The user will be able to fill a form setting up the historical event, language, location, and a search query.

Jun. 2020-Aug. 2020

Wikimedia Commons Upload Pilot Project: American Currency Collection

Harvard Library, Digital Strategies & Innovation

GitLab Repo

Using the Agile/Scrum protocol develop an interface that interacts and download metadata & media from the Harvard LibraryCloud API and uploads it to the Wikimedia Commons. Performed a metadata crosswalk from an API with an XML/MODS schema to a Wikimedia Resource template. Also, created a workflow and robust documentation for future projects.
